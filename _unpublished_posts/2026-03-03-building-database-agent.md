## Introduction

In this course, we learn how to build our own AI agent to interact with tabular data and SQL databases, using the LangChain agents framework.

Learnings from this course:

- Deploy LLMs to build your AI agent
- Implement RAG with tabular data
- Develop your own database agent
- Build a function calling system
- Integrate Azure OpenAI Assistance API
- Use Azure OpenAI service and LangChain

## Your First AI Agent

The road to generative AI: Artificial Intelligence -> Machine Learning -> Deep Learning -> Generative AI.

Generative AI:

- Foundation models (multiple skills and applications in one single model)
- Different data formats
- Natural language instead of programming

Ways to customize LLMs:

- RAG / Grounding
- Fine tuning

## Interacting with CSV data

Use Langchain's dataframe agent to interact with CSV data.

## Connecting to a SQL database

Use Azure OpenAI Function Calling feature to make the LLM output more reliable and predictable.

## Leveraging Assistant API for SQL databases

Assistants API is being deprecated. Use [Responses API](https://developers.openai.com/api/reference/responses/overview) instead for generating model responses.
