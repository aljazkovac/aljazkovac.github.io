# Good Flashcard Principles

## Atomic Knowledge

Each card should test exactly one fact. If a card has multiple parts, split it.

## The Minimum Information Principle

The cards you create must be as simple as possible. Simple cards are easier to review and easier to remember.

## Use Cloze Deletion for Context

Cloze deletions are great for learning technical workflows, syntax, or sentences where context is key.

## Question/Answer (Basic) for Definitions

Use Basic cards for "What is X?" or "List the 3 types of Y" (if the list is short).

## Avoid Lists

If you must learn a list, use multiple Cloze deletions or individual Basic cards. Avoid cards that ask "List all 10 principles of X."

## Add "Extra" Information

Always use the third field in the TSV for "Extra" or "Context" to provide more depth when the card is revealed.
